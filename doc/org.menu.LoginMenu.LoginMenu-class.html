userLogon(self, e)

source code 

Fetchs username and password from entry widget.

With database, check if password corresponds to user.

If they don't match, it opens the error window.

If the username and password match, login to the game, start game with the username, destroy login window, unbind keys.

Parameters:
  • e - Passed by keybinding, not actually used.
